PROJECT OVERVIEW
--------------------------------------------------------------------------------
This project investigates the efficacy of multimodal inputs -- combining hand
tracking and voice commands -- as replacements for traditional handheld
controllers in a VR cooking simulator. The goal is to evaluate whether natural
user interfaces (NUI) improve immersion, comfort, and task efficiency compared
to standard controller-based interaction.
